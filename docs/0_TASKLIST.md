# Lore Weaver - 交互式多 Agent 架构细化升级计划 (Step 1.1 ~ Step 3.3)

## 第 1 阶段：全栈细化与微任务网关 (Step 1.1 - 1.2)
- [ ] 1.1 **[Step 1.1: IP DNA 与经济系统编制]**：
  - 更新 FastAPI 后端，基于 `step 1.1_IP_DNA_And_Economy_Prompt.md` 自定义设计提示词模版。
  - 支持生成并存储 IP 特征 assertions、3-Resource 经济系数以及 6-12 幻境境界。
- [ ] 1.2 **[Step 1.2: 12 重天核心剧情大纲规划]**：
  - 承接 1.1 生成的经济系统，调用大纲设计模型生成 12 Node 大纲（含解锁境界、关卡设定、惩罚标准与奖励）。
  - 支持单独对此节点进行 Chat 互动微调，并由用户 "Approve" 确认。

## 第 2 阶段：物理沙盒及代码脚手架重塑 (Step 2.1 - 2.2)
- [ ] 2.1 **[Step 2.1: 系统级脚手架解耦配置]**：
  - 生成并在工作物理沙盒路径自动部署标准 `package.json`, `vite.config.js` 以及 Phaser 核心宿主入口 `js/main.js`。
  - 标准化 ScaleFIT (720x1280) 分辨率锁框架。
- [ ] 2.2 **[Step 2.2: 状态机与持久化仓储自动注入]**：
  - 自动在沙盒注入存储类接口代码 `store.js` 与静态数据中心注册表 `data.js`，预编译技能与装备配方。

## 第 3 阶段：关卡工厂编译、音频合成与审计系统 (Step 3.1 - 3.3)
- [ ] 3.1 **[Step 3.1: 关卡玩法工厂代码高保真生成]**：
  - 结合 JSON 配置，自动编制 12 关物理代码逻辑（落笔为 JavaScript 文件）。
  - 严格注入 transitionLock 与 shutdown 释放机制，杜绝游戏引擎内存泄漏与重复挂载异常。
- [ ] 3.2 **[Step 3.2: 视听及 ASMR 声音合成润色]**：
  - 调用极具艺术美感的声音发生器模块合成 ASMR 音色（正弦、锯齿、脉冲波模拟风暴或碰撞电音），实现纯物理机制声音发生。
  - 注入屏幕震动微动作、伤害漂浮词条特效（VFX Floaters），自动实现 Chinese wordWrap 最优排版。
- [ ] 3.3 **[Step 3.3: 审计质量闸门与微知识结晶回流]**：
  - 执行仿真 E2E 与 VLM 极度异常检查，自动化去 IP 化并沉淀问题知识进入全局优化 Prompts 回流库。

## 第 4 阶段：前端人机干预 UI 体验全面提升
- [ ] 4.1 **重构主流水线进程图**：从宏观六步转变为高保真的 1.1 ~ 3.3 推进 Timeline 指示。
- [ ] 4.2 **开发分段提交与“神识共鸣”(Human-in-the-Loop Dialog)**：用户随时能查看当前微步骤的运行详情、暂挂情况并一键同意或补充精妙意见。

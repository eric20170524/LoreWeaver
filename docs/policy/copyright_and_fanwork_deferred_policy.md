# Copyright and Fanwork Deferred Policy

> Current scope: personal/private use. Public export, distribution, marketplace upload, or hosted sharing must run this checklist first.

## Current Boundary

- `minigame/` may remain a private case library containing real fan-game experiments.
- `LoreWeaver/docs/gameplay_cards/` should describe mechanics and provenance, not copy protected expression.
- `minigame_master/core/lib/` must stay de-IP: no character names, source plot text, protected assets, or project-specific story strings.
- `minigame_master/core/demo/` must use generic labels and generated shapes only.

## Export Cleanup Checklist

Before any public export:

1. Replace specific character/place/faction names with original or generic equivalents.
2. Remove direct quotes and long paraphrases from source fiction.
3. Replace source-matching story beats with original scenario framing.
4. Remove external images/audio/fonts unless license and attribution are documented.
5. Run content safety scan on export surfaces.
6. Review generated manifest, docs, UI strings, and demo code manually.

## Allowed For Now

- Private notes that mention source project paths for engineering provenance.
- Mechanics extraction such as timing windows, collision loops, bridge contracts, reward shapes, and lifecycle lessons.
- Local-only demos that use generic placeholders.

## Not Allowed For Public Export

- Source character names or distinctive faction/location names in generated playable exports.
- Direct novel lines, lyrics, or long recognizable passages.
- Copied images/audio/video or derivative assets without documented rights.
- Marketing copy implying official affiliation.

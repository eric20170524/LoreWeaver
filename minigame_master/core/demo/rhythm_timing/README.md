# rhythm_timing core demo

Standalone Phaser demo for the `rhythm_timing` Gameplay Card (`TapReactionAdapter`).

```bash
# from repo root
npx vite --config minigame_master/core/demo/rhythm_timing/vite.config.mjs --host 127.0.0.1 --port 5190

# theme skins
# ?theme=temple (default) | ?theme=neon
# ?goalValue=8&durationSec=40
```

```bash
npm run check:rhythm-e2e
```

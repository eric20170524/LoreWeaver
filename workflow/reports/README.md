# Gate Reports

This directory stores the latest local gate reports produced while hardening LoreWeaver as an AI-assisted personal game workbench.

- `build_gate_latest.json`: TypeScript/build checks for LoreWeaver and the survivor horde demo.
- `scene_hygiene_latest.json`: static lifecycle and cleanup checks for core gameplay code.
- `runtime_e2e_survivor_horde_latest.json`: latest browser E2E observation for the survivor horde demo.
- `content_safety_scan_latest.json`: export-surface scan for hard-coded source IP terms.
